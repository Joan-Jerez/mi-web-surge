Hecho para la práctica de Integración Continua con GitHub Actions + Surge.

Cómo funciona

  1. Haces commit a index.html en GitHub.
  2. Se dispara el workflow en la rama main.
  3. Actions instala surge y publica el contenido.
  4. Los secrets SURGE_LOGIN y SURGE_TOKEN evitan exponer credenciales.

Tip rápido

Para probar el redeploy, cambia cualquier texto, haz Commit y revisa la pestaña Actions.
